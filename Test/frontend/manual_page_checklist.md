# 前端后备检查清单

默认前端验证使用 WebdriverIO 自动化。本清单只用于自动化失败后的人工复核或课堂演示，不作为默认测试结果中的待办项。

## 无法完全自动化项

- 云存储下载按钮会打开 Tauri 原生保存对话框，当前不引入 `tauri-plugin-wdio` 或测试专用保存路径，因此 UI 下载流程不自动化。下载文件内容一致性应由 API/Shell 测试覆盖。
- Tauri E2E 当前使用业务代码硬编码的 8080 端口。若需要独立临时端口，需批准 Rust/Tauri 最小重构。
- 完整前端 console 与 Rust backend log forwarding 需要 `tauri-plugin-wdio`，当前不改业务入口。

## 复核内容

- 登录/注册/退出。
- 手动备份与任务列表状态。
- 备份还原成功与错误密码失败。
- 实时监听启动/停止按钮状态。
- 定时任务列表与停止操作。
- 备份管理列表、metadata 展示和淘汰结果。
- 云存储上传、列表、删除。
